Politique de sécurité — Lokizio

Lokizio prend très au sérieux la sécurité de sa plateforme et de ses utilisateurs. Si tu identifies une vulnérabilité, ce document explique comment la signaler.

📩 Comment signaler

Envoie un email à fabien65400@hotmail.fr (ou security@lokizio.com quand le domaine sera actif) avec :

  • Une description claire du problème
  • Les étapes pour reproduire
  • L'impact potentiel (qui est touché, gravité)
  • Ta version du navigateur + l'environnement
  • Optionnel : ton nom/handle pour mention publique

🟢 Périmètre (in scope)

  • ✓ https://fabienlacaze.github.io/lokizio/ — l'app PWA
  • ✓ Edge Functions Supabase https://mrvejwyvhuivmipfwlzz.supabase.co/functions/v1/*
  • ✓ Toutes les RLS policies de la DB Supabase Lokizio
  • ✓ Le flux Stripe Connect (côté Lokizio uniquement)

🔴 Hors périmètre (out of scope)

  • ✗ Stripe — reporter à security@stripe.com
  • ✗ Plateforme Supabase — reporter à security@supabase.io
  • ✗ Sentry, jsDelivr, autres dépendances tierces
  • ✗ Spam / phishing visant les utilisateurs (signaler au support)
  • ✗ DDoS / attaques par déni de service (ne pas tester sans accord)

⏱ Engagements Lokizio

  • Accusé de réception sous 48h ouvrables
  • Première évaluation sous 7 jours
  • Patch déployé selon la sévérité :
    • Critique (RCE, leak de données, compromise auth) : 72h
    • Élevé (XSS, IDOR, escalation) : 7 jours
    • Moyen / Bas : 30 jours
  • Mention publique de ton signalement (avec ton accord)

🤝 Engagements du chercheur

  • Ne PAS exploiter ni divulguer la vulnérabilité avant le patch
  • Ne PAS dégrader le service (pas de tests destructifs en prod)
  • Ne PAS accéder à des données d'autres utilisateurs au-delà du strict minimum pour démontrer le bug
  • Patienter au moins 90 jours avant divulgation publique

💼 Bug bounty

Lokizio n'a pas (encore) de programme de récompenses monétaires. Les signalements responsables sont remerciés par :

  • Mention dans le Hall of Fame sécurité (avec ton accord)
  • Crédit dans les notes de version (changelog)
  • Lokizio Premium gratuit à vie (pour vulnérabilités sérieuses)
